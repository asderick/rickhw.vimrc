# Install

* Please backup your vim settings before install.
* The `install.sh` will backup existing settings as bellow
    1. `~/.vim/` to `~/.vim.bak-1`
    2. `~/.vimrc` to `~/.vimrc.bak-1`


```
$ git clone https://github.com/rickhw/rickhw.vimrc
$ cd rickhw.vimrc
$ ./install.sh
```
